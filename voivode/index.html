Voivode

Lead the shadows. Command the monsters. Shape the battlefield.

  • Role: Programmer
  • Genre: 3D Real-Time Tactics
  • Engine: Unity Engine
  • Plattform: PC
  • Duration: 4 months (Verticle Slice)

Voivode is a 3D real-time tactics game set in 15th-century Wallachia, where players command vampires and werewolves to repel Ottoman invaders. The game challenges players to master stealth, strategy, and battlefield management. Each monster brings unique abilities to the fight, blending shadowy ambushes with brutal assaults.

Taking inspiration from classic titles like Commandos and Desperados, players must adapt quickly, using smart positioning, tactical decision-making, and teamwork to overcome increasingly complex scenarios. Enemy AI System

I developed an AI system for Voivode using a finite state machine architecture. Enemies patrol predefined routes, engage the player upon detection, and search the environment if they lose sight of their target. The VFX effects of the enemies were also created by me.

Save & Load System

I implemented a JSON-based save and load system, accessible through the in-game pause menu. Players can save their progress at any time, preserving the complete game state including player position, objectives, and enemy behaviors. When reloading, the game seamlessly restores the session, ensuring smooth continuation without frustrating restarts.

In addition, I implemented several VFX elements and the camera control system.
